# [Project] LG 가전 소비자 분석 및 신규 시나리오 발굴
> **LG 가전(의류관리기/청소기) 사용자 리뷰 데이터 분석을 통한 UX 개선 및 AI 기능 제안**

본 프로젝트는 네이버 카페, 블로그, 유튜브 등 다양한 채널에서 수집된 소비자 리뷰를 바탕으로, 실제 사용자의 이용 패턴을 분석하고 제품에 반영될 수 있는 신규 기능 및 시나리오를 도출하는 것을 목적으로 합니다.

---

## 1. 프로젝트 개요
* **분석 대상**: LG 스타일러(의류관리기), 무선스틱청소기, 로봇청소기
* **데이터 규모**: 약 50,000건 이상의 소비자 리뷰 (네이버, 유튜브, 티스토리 등)
* **핵심 분석**:
    - AI 모델과 일반 모델 간의 사용자 인식 차이 분석
    - 연도별 소비자 핵심 키워드 변화 추이 파악
    - 키워드 네트워크 분석을 통한 실제 사용 시나리오 도출

## 2. 기술 스택 (Tech Stack)
* **Language**: `Python`
* **Environment**: `Jupyter Notebook`
* **Library**:
    - **NLP**: `Konlpy(Okt)`, `re` (정규표현식 전처리)
    - **Analysis**: `Pandas`, `Numpy`, `NLTK (N-gram)`
    - **Visualization**: `Matplotlib`, `Seaborn`, `WordCloud`, `NetworkX`

## 3. 주요 분석 프로세스
### 1) 데이터 전처리 (Data Preprocessing)
- 광고성 게시글, 중복 데이터, 질문글 등 노이즈 데이터 제거 (300자 이하 짧은 글 필터링 등)
- 형태소 분석기(Okt)를 활용하여 명사, 동사, 형용사 중심의 토큰화
- 가전 맞춤형 불용어(Stopwords) 정의 및 처리

### 2) 데이터 분석 기법
- **빈도 분석**: 연도별/브랜드별 주요 키워드 점유율 비교
- **N-gram 분석**: 특정 기능(예: '살균', '스팀')과 함께 언급되는 사용자 감정 및 맥락 파악
- **네트워크 분석**: `NetworkX`를 활용하여 키워드 간의 상관관계를 시각화하고 사용자 이용 행태 군집화

## 4. 핵심 인사이트 (의류관리기 중심)
- **사용자 시나리오 도출**: '패딩 건조', '아기옷 살균', '요가복 관리' 등 특정 의류 소재와 연결된 전용 코스에 대한 니즈 확인
- **AI 기능 인식**: 'AI 바닥 인식', '자동 흡입력 조절' 등 지능화된 기능이 사용자 편의성에 미치는 긍정적 영향 확인
- **Pain Point**: 특정 소음이나 관리의 번거로움에 대한 언급 패턴 분석을 통해 개선점 도출

## 6. 프로젝트 성과 및 기대효과
- 실제 사용자 데이터를 근거로 한 **'다운로드 코스' 세분화 전략** 제안
